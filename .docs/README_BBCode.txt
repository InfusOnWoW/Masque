[COLOR="Orange"][SIZE="4"]About[/SIZE][/COLOR]

[I]Masque[/I] is an add-on for [URL="https://worldofwarcraft.com"]World of Warcraft[/URL] that provides a skinning engine for button-based add-ons. Note that you [B]must[/B] have an add-on that supports [I]Masque[/I] installed in order to use it. [I]Masque[/I] does [B]not[/B] support the default interface.

[COLOR="Orange"][SIZE="4"]Installation[/SIZE][/COLOR]

1. Download the package from one of the following sites:
   [LIST]
   [*] [URL="https://mods.curse.com/addons/wow/masque"]Curse[/URL]
   [*] [URL="https://www.wowace.com/projects/masque"]WoW Ace[/URL]
   [*] [URL="http://www.wowinterface.com/downloads/info12097"]WoW Interface[/URL]
   [/LIST]
2. Extract the package to your [B]World of Warcraft\Interface\AddOns[/B] directory.

[COLOR="Orange"][SIZE="4"]Options[/SIZE][/COLOR]

To access [I]Masque[/I]'s options, use the [B]/msq[/B] or [B]/masque[/B] chat commands. You can also access the options via the [I]Interface Options Window[/I] or by clicking on [I]Masque[/I]'s icon in a [I]DataBroker[/I] display.
